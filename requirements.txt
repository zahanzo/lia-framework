# ============================================================
# AI Personal Assistant Framework - Dependencies
# ============================================================
# Recommended Python: 3.10.11
# Python 3.11+ may have issues with PyTorch / faster-whisper.
# Download: https://www.python.org/downloads/release/python-31011/
#
# QUICK START:
#
#   1. Core (always required):
#      pip install -r requirements.txt
#
#   2. Pick a voice engine (choose one):
#      pip install edge-tts                          <- default, online, free
#      pip install kokoro soundfile                  <- local, high quality
#      pip install piper-tts                         <- local, language-native
#      pip install elevenlabs                        <- online, premium
#
#   3. Microphone input (required for voice):
#      pip install pyaudio
#
#   4. Pick a speech recognition engine (choose one):
#      pip install faster-whisper torch torchaudio   <- local Whisper (+ pyaudio)
#      pip install SpeechRecognition                 <- Google STT fallback (+ pyaudio)
#
#   5. Optional features:
#      pip install sentence-transformers numpy       <- semantic memory / RAG
#      pip install mss                               <- screen capture / vision
#      pip install pyautogui                         <- GUI automation in MCP tools
#      pip install psutil pywin32                    <- meeting detection (Windows)
#      pip install websockets                        <- VTube Studio lip sync
# ============================================================
